# Random Quote Generator

**Автор:** [сидорова ]
**Язык:** Python 3
**Библиотеки:** Tkinter, random, json

## Описание

Приложение для генерации случайных цитат с возможностью фильтрации по автору и теме. История сгенерированных цитат сохраняется в JSON и отображается в отдельном списке.

## Возможности

- Генерация случайной цитаты.
- Фильтрация по автору и теме.
- Сохранение истории в JSON.
- Проверка корректности ввода.
- Простой и интуитивный интерфейс.

## Запуск

1. Установите Python 3.
2. Клонируйте репозиторий.
3. Запустите `main.py`.
4. (Опционально) Добавьте свои цитаты в `quotes.json`.

## Примеры использования

- Нажмите «Сгенерировать цитату» — появится случайная цитата.
- Введите автора или тему в поля фильтра и нажмите «Применить фильтр».
- История отображается в отдельном списке.
